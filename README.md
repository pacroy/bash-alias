# Bash Aliases

## Aliases

Append `alias.sh` to your `.zshrc` or `.bashrc` or `.bash_profile` and reload it.

```
curl -s https://raw.githubusercontent.com/pacroy/bash-alias/master/alias.sh >> ~/.bashrc && source ~/.bashrc
```
