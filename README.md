# Bash Aliases

## Installing alias

```
curl -sS https://raw.githubusercontent.com/pacroy/bash-alias/master/install_alias.sh | bash -
```

## Installing tmux

```
curl -sS https://raw.githubusercontent.com/pacroy/bash-alias/master/install_tmux.sh | bash -
```

## Updating alias

```
update_alias
```

## Updating tmux

```
update_tmux
```
